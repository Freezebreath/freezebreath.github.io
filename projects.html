Game Length

An unofficial native Android application for the website HowLongToBeat.com. Uses libraries such as jsoup and ion to asynchronously download and display data. Users can save games to their favorites to view offline and to be notified of any updates via a background service. Available on the Amazon App Store. View the code on GitHub.

SixSigma Game Store

A fully featured ASP.NET MVC game store. Features include: user profiles, wishlist, friends list, events, reports, order management, game import and checkout. Game information for import is provided by TheGamesDB API. Payments are processed using Stripe.

Tic-Tac-Toe & Tic-Tac-Toe Manager

A pair of Android applications. Tic-Tac-Toe: a Java port of a previous C# Tic-Tac-Toe game to the Android platform with additional features. Tic-Tac-Toe Manager: a tool which allows users manage players and games and view statistics.

Bird Hunt

A Duck Hunt-style game built using XNA which tasks players to shoot birds (and bees) as efficiently as possible to conserve ammo and earn points toward a high score.

SwipeTask

A simple, cloud-enabled todo app for Android with an easy to use interface. Uses the Parse app platform. Available on the Google Play Store.

Airline Reservations

A C# WinForms application using MetroFramework in order to provide a beautiful, modern user interface.

Team Manager

A hybrid mobile application which allows coaches to create and manage teams, players, games and other coaches. Built using jQuery mobile and compiled to iOS and Android with native features such as the camera via Apache Cordova/PhoneGap.
